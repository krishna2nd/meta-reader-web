## META SCRAPER 

An ui implementation for meta reader library 
[meta-reader](https://github.com/krishna2nd/meta-reader) [![NPM Version][npm-image]][npm-url]

### View Demo
  * clone git repository
  * Run `npm install`
  * Run `npm start`
  * Access `http://localhost:8080/` in any of the browser
